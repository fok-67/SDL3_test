# SDL3 test
<p>A very simple SDL3 test.</p>

## Getting Started
<p>Use CMake to compile the sources.</p>

### Prerequisites
<p>It works on Ubuntu 22.04, OpenSuse Leap 16 and Windows 11. You need the C++ development tools for your platform and SDL3.</p>

### Built With
* CMake and gcc 15 on Linux
* CMake and MSVC 14 (Visual Studio 2026) on Windows 11
